# 제품 요구사항 정의서 (PRD): AI 서비스 조합 추천 웹사이트

## 1. 제품 개요 (Product Overview)
**서비스 명 (가칭):** AI Stack Recommender (AI 테크스택 추천기)
**목적:** 사용자가 만들고자 하는 프로젝트(웹, 앱, 서비스 등)의 아이디어나 요구사항을 입력하면, 이를 구현하는 데 가장 적합하고 효율적인 인공지능(AI) 서비스 및 도구 조합을 추천해주는 웹서비스.
**타겟 사용자:** 
- AI 기술을 활용해 서비스를 구축하고자 하는 개발자 및 기획자
- 어떤 AI API나 도구를 써야 할지 막막한 초보 창업가
- 프로젝트 개발 생산성을 극대화하고자 하는 메이커(Maker)

## 2. 핵심 기능 (Key Features)

### 2.1. 프로젝트 요구사항 입력
- **자연어 프롬프트 입력:** "사용자의 얼굴을 인식해서 비슷한 연예인을 찾아주는 앱을 만들고 싶어"와 같이 사용자가 생각하는 아이디어를 자연어로 쉽게 입력.
- **선택형 필터 옵션:** 예산(무료/유료), 기술 수준(노코드/로우코드/전문개발), 타겟 플랫폼(Web, iOS, Android) 등 부가적인 제약 사항 설정 가능.

### 2.2. 맞춤형 AI 테크 스택 추천 엔진
- 입력된 프로젝트 정보를 분석하여 기능별로 최적의 AI 도구를 매칭.
- 예: LLM(텍스트 생성), Computer Vision(이미지 처리), Voice(음성 인식/합성), 생산성(코딩 보조, 기획 보조) 등 카테고리별 추천.

### 2.3. 상세 추천 결과 제공
- **추천 이유:** 왜 이 AI 도구(예: OpenAI API, Claude, Midjourney, Supabase 등)가 해당 프로젝트에 적합한지 상세 설명.
- **예상 비용 및 라이선스:** 무료 플랜 여부, 대략적인 API 호출 비용, 오픈소스(상업적 이용 가능 여부) 정보 안내.
- **러닝 커브 및 관련 문서:** 해당 기술을 익히기 위한 난이도와 공식 문서 링크 제공.

### 2.4. 결과 저장 및 공유
- 추천받은 테크 스택 조합을 PDF로 다운로드하거나 링크로 팀원들과 공유할 수 있는 기능.

## 3. 사용자 여정 (User Journey)
1. **랜딩 페이지:** 서비스의 가치 제안과 직관적인 프롬프트 입력창을 마주함.
2. **정보 입력:** 프로젝트 아이디어와 현재 자신의 개발 숙련도, 예산 등을 입력.
3. **분석 단계 (로딩):** "입력하신 프로젝트를 분석 중입니다...", "최적의 AI 스택을 탐색 중입니다..." 등의 애니메이션 제공.
4. **결과 확인:** 카테고리별로 깔끔하게 정리된 AI 추천 리스트와 다이어그램 형태의 아키텍처 가이드 확인.
5. **액션:** 추천된 도구들의 공식 사이트로 이동하거나 결과를 저장 및 공유.

## 4. 비기능적 요구사항 및 기술 스택 (Technical Requirements)
- **Frontend:** React, Next.js, Tailwind CSS (반응형 및 모던 UI 구현)
- **Backend:** Node.js, Python (FastAPI/Django) - 추천 로직 및 프롬프트 처리
- **Core AI:** OpenAI API (GPT-4 등) 또는 Anthropic Claude API (자연어 요구사항 분석 및 추천 알고리즘 설계용)
- **Database:** PostgreSQL 또는 MongoDB (사용자 입력 데이터 및 통계 저장용)

## 5. 단계별 출시 계획 (Roadmap)
- **Phase 1 (MVP):** 핵심 기능인 자연어 입력 기반 AI 도구 텍스트 추천 리스트 제공.
- **Phase 2 (기능 고도화):** 카테고리별 상세 필터링, 관련 아키텍처 다이어그램 임베딩 제공, 사용자 리뷰 및 평점 도입.
- **Phase 3 (비즈니스 모델):** 특정 AI 서비스들의 제휴/레퍼럴 링크 연동, 엔터프라이즈용 맞춤 컨설팅 리포트 자동 생성 기능.
